Boards

Resource Description
GET /api/v1/boards JWT PROTECTED: Get all users boards
GET /api/v1/boards/:id JWT PROTECTED: Get user board
POST /api/v1/boards JWT PROTECTED: Create new board
PUT /api/v1/boards/:id JWT PROTECTED: Update board
DELETE /api/v1/boards/:id JWT PROTECTED: Delete board
PUT /api/v1/boards/:id/add_user JWT PROTECTED: Add user to board
DELETE /api/v1/boards/:id/add_user JWT PROTECTED: Remove user from board
